# Định nghĩa
Đôi khi, khi xuất hiện dữ kiện mới, xác suất sẽ bị thay đổi. Vậy xác suất của biến cố A với điều kiện biến cố B đã xảy ra là 
$$P(A|B) = \dfrac{P(A \cap B)}{P(B)}, \quad P(B)>0$$
# Tính chất
- $0 \leq P(A|B) \leq 1$
- $P(B|B) = 1$
- Nếu $A.C = \emptyset$ thì $P[(A+C)|B] = P(A|B) + P(C|B)$
- $P(\bar{A}|B) = 1 - P(A|B)$
